1. Встанови Python 3.10 або новіший.
2. У цій папці відкрий термінал.
3. Встанови бібліотеку командою:
   pip install -r requirements.txt
4. Відкрий файл token.txt
5. Видали текст PASTE_YOUR_BOT_TOKEN_HERE і встав свій токен бота.
6. Запусти бота командою:
   python bot.py
7. Якщо бот запустився, у консолі буде:
   ✅ Бот запущений як ...

Команди в Discord:
!help
!balance
!give
!open
!inventory
!cards

Додавання картки для адміністратора:
!addcard Naruto 20 rare https://link.png

Якщо картинки немає, напиши так:
!addcard Naruto 20 rare none
